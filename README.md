# 📒 GitHub Planner (Markdown)

이 문서는 GitHub 저장소에 올려서 사용하기 좋은 마크다운 플래너 템플릿입니다. **파일명을 `Planner.md` 또는 원하는 이름으로 저장**한 뒤 GitHub에서 열어보면 바로 사용할 수 있습니다.

---

## 사용법

* 이 파일을 복제하여 개인 저장소에 넣고, 매주/매일 복사해서 사용하세요.
* 체크박스(`- [ ]`)를 클릭하면 GitHub에서 체크가 가능합니다.
* `##` 헤더 단위를 복사해서 주간/일간 템플릿으로 사용하세요.
